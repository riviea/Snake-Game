- 방향키를 눌러서 뱀(◆)을 조작합니다.
- 자신의 몸통, 혹은 외벽(■)과 부딪히면 게임오버됩니다.
- 음식(◎)을 먹으면 게임의 속도가 빨라집니다.
- 게임오버 시 프로그램이 종료됩니다.
